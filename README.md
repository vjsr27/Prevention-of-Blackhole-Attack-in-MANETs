# Prevention of Blackhole Attack in MANETs

## Overview
This project focuses on improving the security of Mobile Ad-hoc Networks (MANET) by designing an Active Authentication Scheme (AAS). The proposed model protects the network from various routing attacks and improves overall network reliability.

## Key Features
- Detection and prevention of blackhole attacks in MANET
- Protection against selective forwarding and false routing attacks
- Defense against route spoofing and malicious nodes
- Improved packet delivery and network stability

## Technologies Used
- Network Security Concepts
- MANET Routing Protocols
- Authentication Mechanisms
- Simulation-based Network Analysis

## Results
The proposed AAS model improves packet delivery by **33.9%** and enhances network reliability even in the presence of malicious nodes.
